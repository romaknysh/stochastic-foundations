# Assignment 06 — Dice Rolling: PDF and Expected Value

## Problem
Roll a die up to 3 times, stopping when an even number appears.
Record the value on the last roll. Compute PDF and expected values.

## Random Variables
| Variable | Description                        |
|----------|------------------------------------|
| Y        | Number of rolls needed (1, 2, or 3)|
| X        | Recorded die value                 |

## Results
| Variable | Expected Value |
|----------|----------------|
| E(Y)     | 1.82           |
| E(X)     | 6.12           |
