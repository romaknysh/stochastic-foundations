# Assignment 09 — Distribution Comparison

## Problem
Compare four discrete probability distributions,
compute key statistics and confidence intervals.

## Distributions
| Distribution | EX | DX  | Std Dev |
|--------------|----|-----|---------|
| Uniform      | 20 | 140 | 11.83   |
| Binomial     | 20 | 10  | 3.16    |
| Poisson      | 20 | 20  | 4.47    |
| Geometric    | 20 | 420 | 20.49   |

## Statistics per Distribution
- Expected value E(X)
- Variance D(X)
- Standard deviation σ
- Confidence interval [EX - cσ, EX + cσ]
- Count of values within interval

## Key Observation
Same mean (EX=20), very different spread:
Geometric >> Uniform >> Poisson >> Binomial
