# Assignment 08 — Sum of Dice vs Binomial Distribution

## Problem
Roll 5 dice (each 0–9), compute their sum.
Compare the empirical PDF of the sum with a theoretical Binomial fit.

## Parameters
| Parameter     | Value |
|---------------|-------|
| n_dice        | 5     |
| die_faces     | 0–9   |
| n_simulations | 36    |

## Charts
- `PDF` — empirical distribution of the sum
- `PDF a Binom` — empirical vs theoretical Binomial overlay

## Result
Empirical PDF approximates Binomial shape,
peaking around sum = 20–23
