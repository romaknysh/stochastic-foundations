# Lab 02 — Score Distribution Combinations

## Problem
Count valid ways to distribute points using denominations 5, 2, and 0,
given constraints on their counts.

## Constraints
| Variable     | Constraint |
|--------------|------------|
| points (B)   | multiples of 5 (5, 10, 15, 20) |
| fives (p)    | <= 4 |
| twos (j)     | even multiples (0, 2, 4...) |
| zeros (h)    | 0 or 1 |

## Result
| Total combinations | Valid combinations |
|--------------------|-------------------|
| 22                 | 9                 |
