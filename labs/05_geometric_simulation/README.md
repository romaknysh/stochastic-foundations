# Lab 05 — Geometric Distribution Simulation

## Problem
Simulate rolling a fair die until a 6 appears.
Estimate the mean number of rolls required.

## Method
Roll-by-roll log — each trial records every roll until 6 lands.

## Result
mean_trial_length ≈ 7.79 rolls

## Theoretical Expected Value
E(X) = 1 / P(6) = 1 / (1/6) = 6
